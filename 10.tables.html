tables

In HTML, tables are used to display data in a structured format, using rows and columns. Here's a breakdown of the key elements and attributes involved.

Key Table Tags:
table - Defines the table structure
tr - Defines a table row
th - Defines a table header cell (bold and centered by default)
td - Defines a table data cell
thead - Groups the header content
tbody - Groups the body content
tfoot - Groups the footer content
caption - Adds a title or description to the table
colgroup - Specifies column formatting
col Defines - column properties within colgroup
