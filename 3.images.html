Images

In HTML, images are added using the (img) tag, which is self-closing and doesn't have a closing tag.
for example:img src="logo.png" alt="Company Logo" width="200" height="100".

src : Specifies the path to the image. It can be relative or absolute.
alt : Provides alternative text for accessibility and SEO.
width and height: Sets the image size in pixels.
